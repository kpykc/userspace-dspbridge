\subsection{Framework Components Data Structures}
Here are the data structures with brief descriptions:\begin{CompactList}
\item\contentsline{section}{\hyperlink{struct_a_c_p_y3___params}{ACPY3\_\-Params} (DMA transfer specific parameters. Defines the configuration of a logical channel )}{\pageref{struct_a_c_p_y3___params}}{}
\item\contentsline{section}{\hyperlink{struct_d_m_a_n3___params}{DMAN3\_\-Params} (The module configuration structure for DMAN3 implementation. \item\contentsline{section}{\hyperlink{struct_i_a_l_g___fxns}{IALG\_\-Fxns} (Defines the fields and methods that must be supplied by all x\-DAIS algorithms )}{\pageref{struct_i_a_l_g___fxns}}{}
\item\contentsline{section}{\hyperlink{struct_i_a_l_g___mem_rec}{IALG\_\-Mem\-Rec} (Memory records )}{\pageref{struct_i_a_l_g___mem_rec}}{}
\item\contentsline{section}{\hyperlink{struct_i_a_l_g___obj}{IALG\_\-Obj} (Algorithm instance object definition )}{\pageref{struct_i_a_l_g___obj}}{}
\item\contentsline{section}{\hyperlink{struct_i_a_l_g___params}{IALG\_\-Params} (Algorithm instance creation parameters )}{\pageref{struct_i_a_l_g___params}}{}
\item\contentsline{section}{\hyperlink{struct_i_a_l_g___status}{IALG\_\-Status} (Pointer to algorithm specific status structure )}{\pageref{struct_i_a_l_g___status}}{}
\item\contentsline{section}{\hyperlink{struct_i_d_m_a3___channel_rec}{IDMA3\_\-Channel\-Rec} (DMA Channel Descriptor to logical DMA channels )}{\pageref{struct_i_d_m_a3___channel_rec}}{}
\item\contentsline{section}{\hyperlink{struct_i_d_m_a3___fxns}{IDMA3\_\-Fxns} (These fxns are used to query/grant the DMA resources requested by the algorithm at initialization time, and to change these resources at runtime. All these fxns are implemented by the algorithm, and called by the client of the algorithm )}{\pageref{struct_i_d_m_a3___fxns}}{}
\item\contentsline{section}{\hyperlink{struct_i_d_m_a3___mem_rec}{IDMA3\_\-Mem\-Rec} (Record containing attributes of the IDMA3 Channel environment memory when (optionally) the IDMA3 Channel is requested with a non-NULL \hyperlink{struct_i_d_m_a3___protocol_obj}{IDMA3\_\-Protocol\-Obj} )}{\pageref{struct_i_d_m_a3___mem_rec}}{}
\item\contentsline{section}{\hyperlink{struct_i_d_m_a3___obj}{IDMA3\_\-Obj} (\hyperlink{struct_i_d_m_a3___obj}{IDMA3\_\-Obj} holds the private state associated with each logical DMA channel )}{\pageref{struct_i_d_m_a3___obj}}{}
\item\contentsline{section}{\hyperlink{struct_i_d_m_a3___protocol_obj}{IDMA3\_\-Protocol\-Obj} (These functions are used to determine memory requirements for and initialize the IDMA3 protocol's environment that will be allocated by the DMA manager )}{\pageref{struct_i_d_m_a3___protocol_obj}}{}
\end{CompactList}
