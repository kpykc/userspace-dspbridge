config cacheWritebackInvalidateFxn

//! Name of function implementing cache writeback invalidate with the following signature: (*DSKT2_CacheWBInvFxn)(Ptr blockPtr, size_t byteCnt, bool wait)
 
config String cacheWritebackInvalidateFxn  = "BCACHE_wbInv"

DETAILS

For C6000 platforms DSKT2 calls this function to ensure that the algorithm instance memory is not resident in Cache when it is instantiated. This is necessary for compliance with XDAIS C6000 DMA Rules and Guidelines.
